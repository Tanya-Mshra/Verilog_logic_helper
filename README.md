⚡ Verilog Logic Helper
An AI-powered assistant designed for B.Tech students to generate Verilog hardware modules and testbenches from natural language descriptions.

🛠️ Tech Stack
Frontend: React with Tailwind CSS

Backend: Node.js with Gemini API

AI Model: Gemini 2.5 Flash

🚀 How to Run
Follow these steps to get the project running locally:

1. Clone the repository
Bash
git clone https://github.com/Tanya-Mshra/Verilog_logic_helper.git
cd Verilog_logic_helper
2. Setup Backend
Navigate to the backend folder.

Create a .env file and add your API key: GEMINI_API_KEY=your_key_here.

Run the following commands:

Bash
npm install
npm start
3. Setup Frontend
Open a new terminal and navigate to the frontend folder.

Run the following commands:

Bash
npm install
npm start
